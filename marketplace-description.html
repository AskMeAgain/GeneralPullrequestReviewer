General Pullrequest Reviewer

  • Complete Pullrequest review integration into Intellij
  • See all open pullrequests
  • In IDE review of pullrequests
  • Only Gitlab is supported, but more will come

Getting started

  • Go to settings
  • Tools
  • Pullrequests
  • Select a connection type in the dropdown
  • click "Add Connection"
  • Fill out the needed connection values and apply changes
  • You can now go to the tool window (right tool window side) and see all Projects/Merge requests

Reviewing Mergerequests

You can open any MR and right click anywhere to create merge request comments
