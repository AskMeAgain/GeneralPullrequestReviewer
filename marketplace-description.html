General Pullrequest Reviewer

Review all your merge requests from within Intellij Idea. Create and read merge request discussions, directly look up code and more. Completely for free and includes all popular git integrations (WIP).

  • Complete Pullrequest review integration into Intellij
  • In IDE review of pullrequests
  • Gitlab Integration

Getting started

  • Go to settings
  • Tools -> Pullrequests
  • Select a connection type in the dropdown
  • click "Add Connection"
  • Fill out the needed connection values and apply changes
  • You are ready to review! Open the Pullrequest tool window on the right.
